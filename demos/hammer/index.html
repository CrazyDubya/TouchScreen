Hammer.js Gesture Library

Advanced gesture recognition library for touch interactions

üî® Hammer.js

Hammer.js is a powerful JavaScript library that recognizes touch gestures made by touch, mouse, and pointer events. It's lightweight, has no dependencies, and is perfect for mobile-first development.

Key Features

üëÜ Multi-touch

Recognizes multiple simultaneous touch points and complex gestures like pinch, rotate, and multi-finger swipes.

üéØ Gesture Recognition

Built-in recognizers for pan, pinch, press, rotate, swipe, and tap gestures with customizable thresholds.

‚ö° Lightweight

Only ~7KB minified and gzipped with no dependencies, making it perfect for performance-critical applications.

üîß Customizable

Fine-tune gesture recognition with options for direction, threshold, velocity, and simultaneous gesture handling.

Supported Gestures

  • Pan: Touch and move in any direction
  • Pinch: Two-finger zoom in/out
  • Press: Long press (touch and hold)
  • Rotate: Two-finger rotation
  • Swipe: Quick directional flick
  • Tap: Single or double tap recognition

Installation & Usage

Install via npm

npm install hammerjs

CDN Link

<script src="https://cdn.jsdelivr.net/npm/hammerjs@2.0.8/hammer.min.js"></script>

Basic Example

const element = document.getElementById('myElement');
const hammer = new Hammer(element);

// Listen for tap
hammer.on('tap', (e) => {
  console.log('Tapped!');
});

// Listen for swipe
hammer.on('swipe', (e) => {
  console.log('Swiped in direction:', e.direction);
});

// Listen for pinch
hammer.get('pinch').set({ enable: true });
hammer.on('pinch', (e) => {
  console.log('Pinch scale:', e.scale);
});
